 * <br/>
28 
 * ISAAC (Indirection, Shift, Accumulate, Add, and Count) generates 32-bit
29 
 * random numbers.
30 
 * ISAAC has been designed to be cryptographically secure and is inspired
31 
 * by RC4.
32

34

35

38
